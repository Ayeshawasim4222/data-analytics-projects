# 📊 Sales Data Analysis Project

## 📌 Overview
This project focuses on analyzing sales data to extract meaningful insights such as product performance, city-wise sales, and customer trends.

## 🛠 Tools Used
- Excel (Data Cleaning & Visualization)
- SQL (Data Analysis Queries)
- Python (Basic understanding)

## 📁 Project Files
- `sales_analysis.xlsx` → Raw data + analysis + charts
- `sales_analysis_presentation.pptx` → Project explanation
- `sales_analysis_queries.sql` → SQL queries for analysis

## 📊 Key Insights
- Identified top-selling products
- Analyzed city-wise sales performance
- Calculated average, maximum, and minimum sales
- Product distribution analysis

- Data Cleaning
- Data Analysis
- SQL Query Writing
- Data Visualization
